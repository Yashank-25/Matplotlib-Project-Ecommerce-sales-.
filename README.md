Project Title / Headline:-
E-Commerce Sales Dashboard using Python, Pandas & Matplotlib

Short Description / Purpose:-
Developed an E-Commerce Sales Dashboard using Python, Pandas, and Matplotlib to analyze sales performance across different regions, product categories, and time periods. The project performs Exploratory Data Analysis (EDA) by calculating key business metrics and visualizing insights through various charts, helping identify sales trends, top-selling products, and profit distribution.

Tech Stack:-
Programming Language: Python
Libraries: Pandas, Matplotlib
IDE: Visual Studio Code (VS Code)
Dataset Format: CSV
Data Source
Self-created E-Commerce Sales Dataset (CSV) containing 300 records.
Dataset includes the following fields:
Order_ID
Date
Product
Category
Region
Sales
Profit
Quantity

Features / Highlights:-
Loaded and processed sales data using Pandas.
Performed data cleaning and checked for missing values.
Generated descriptive statistics and dataset summary.
Converted date values for monthly sales trend analysis.
Calculated Total Sales and Total Profit.
Visualized Sales by Region using a Bar Chart.
Analyzed Category-wise Revenue.
Created a Monthly Sales Trend using a Line Chart.
Identified the Top 10 Products by Sales.
Displayed Profit Contribution by Category using a Pie Chart.
Compared Quantity Sold by Category.
Explored the relationship between Sales and Profit using a Scatter Plot.
Compared Profit by Region using a Bar Chart.
